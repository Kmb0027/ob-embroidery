CompTIA A+ 901 Practice Questions

Question 1
Objective: Given a scenario, troubleshoot hard drives and RAID arrays with appropriate tools.
A technician is troubleshooting performance issues with a desktop PC and hears clicking noises coming from inside the case. Which of the following is the MOST likely cause?

A. Low voltage power supply
​B. Failing magnetic hard drive
​C. Corrupted Solid State Drive
D. Overheated RAM

Question 2
Objective: Compare and contrast various WiFi networking standards and encryption types.
A technician has noticed that a client has an unsecured wireless network. Which of the following is the BEST security encryption type to protect the wireless network from intrusion?

A. WEP2
​B. WEP
​C. WPA2
D. WPS

Question 3
Objective: Given a scenario, troubleshoot common problems related to motherboards, RAM, CPU, and power with appropriate tools.
On Monday, a user notices that the PC being used is not keeping correct time. Other users are not having this issue. The user’s PC had a memory upgrade the previous Friday. The user states that the PC was powered off all weekend. Which of the following should the technician check FIRST in troubleshooting this issue?

A. Check the CMOS battery to see if it has been dislodged.
​B. Check that the motherboard is functioning properly.
​C. Check that the DNS name of NTP server has not changed.
D. Check with the administrator to ensure the NTP server is up.

Question 4
Objective: Install and configure laptop hardware and components.
A technician is installing an additional memory module in a laptop. Which of the following components should the technician remove to help mitigate the threat of ESD damage to the DIMM?

A. Hard drive
​B. Battery
​C. CPU
D. Screen
E. Keyboard

Question 5
Objective: Explain the importance of motherboard components, their purpose, and properties.
A technician is building a new PC and notices that it has 8 DIMM sockets labeled 0 through 7. Four are white and four are blue. Which of the following should the technician do FIRST?

A. Install the memory in the four white slots
​B. Install the memory in the four blue slots
​C. Consult the motherboard documentation
D. Install the memory in slots 0 through 3
E. Fill all slots with memory

Question 6
Objective: Given a scenario, troubleshoot and repair common mobile device issues while adhering to the appropriate procedures.
A user’s iPhone will not power on. Which of the following should the technician do FIRST to isolate the problem?

A. Plug the phone into a known good charger
​B. Remove and reseat the battery
​C. Replace the screen glass assembly
​D. Replace the battery and charge the phone

Question 7
Objective: Given a scenario, use appropriate laptop features.
A laptop is setup at a podium for a presentation but is set to extended desktop. The presenter would like it to show exactly what the laptop screen is showing. Which of the following would be the QUICKEST way to accomplish this?

A. Look for the setting in control panel
​B. Toggle the dual display function key
​C. Change the screen orientation
D. Reboot the laptop

Question 8
Objective: Compare and contrast accessories and ports of other mobile devices.
Which of the following connection types can be blocked by a person walking between the two devices that are communicating?

A. Cellular
​B. IR
​C. WiFi
​D. Bluetooth

Question 9
Objective: Compare and contrast types of display devices and their features.
The finance department has hired a new employee who works on payroll data spreadsheets. Which of the following items should the IT department install on the employee’s monitor prior to deploying it?

A. Privacy filter
​B. Screen protector
​C. Spam filter
​D. Monitor stand

Question 10
Objective: Identify the various types of network cables and connectors.
A company upgraded their cabling from CAT5 to CAT6. A technician is tasked with replacing all the patch cords to match the new cabling. The technician has to make patch cords to the proper lengths. The technician is supplied with the CAT6 cable and a crimper but needs to order the proper cable ends. Which of the following should be ordered?

​A. ST
​B. RJ-45
​C. SC
​D. RJ-11

A+ 901 Answer Key

Question 1) B. Failing magnetic hard drive

Question 2) C. WPA2

Question 3) A. Check the CMOS battery to see if it has been dislodged.

Question 4) B. Battery

Question 5) C. Consult the motherboard documentation

Question 6) A. Plug the phone into a known good charger

Question 7) B. Toggle the dual display function key

Question 8) ​B. IR

Question 9) ​A. Privacy filter

Question 10) ​B. RJ-45
